Mii Renderer

This is a tool to render a Mii into image form. It works for any type of Mii - ones created on the Wii, a QR Code, or even a Mii on RiiConnect24's Check Mii Out Channel! The output can be customized however you like. This works great to use as a profile picture!

Check out miicontest.wiilink.ca for more Mii content!
